# OD-Simulator
Reproducible Python workflow for machine-learning outlier detection in rate-normalized pressure data for rate-transient analysis of tight gas reservoirs.

This repository contains the Python code and example datasets used for the study:

**Machine-Learning-Based Outlier Detection for Rate-Normalized Pressure Data in Rate-Transient Analysis of Tight Gas Reservoirs**
The workflow evaluates multiple outlier detection methods for preprocessing rate-normalized pressure (RNP) data. The objective is to improve diagnostic trend identification and linear-flow interpretation in rate-transient analysis (RTA).

## Contents
- Synthetic RNP data generation
- Noise and outlier contamination scenarios
- Outlier detection methods
- Evaluation metrics including accuracy, precision, recall, F1-score, and MAE
- Plotting scripts for reproducing paper figures
- Example workflow notebooks

## Data Notice
The datasets in this repository are synthetic or anonymized examples prepared to support research reproducibility. No confidential, proprietary, or client data are included.
